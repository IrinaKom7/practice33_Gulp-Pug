Introduction Page
1: The menu is controlled by the h1 tags. Simply add your page title and repeat in the menu links.

The images for the slider can be found in the images folder. You can change the styling for this in the css folder - nivostyle.css.
3: You can change the rbbon on the slider, I have provided some options for you in the extras folder. There's also a psd of it.
4: To change it, choose your ribbon and add it to nivostyle.css. If you don't want it, simply delete the ribbon div class.

Services
The services page uses an accordian slider.

2: Each pic has its own data id, starting from 1, continue until you have added all of your pictures, each should have it's own value. To get the images to sort, mix up your classes. You can see how to do this by following my code.

Google Maps:
1: I have used the plugin called jQuery GPS
In order to use the Google Maps API you must have an API key which you can get here: http://code.google.com/intl/en/apis/maps/signup.html
2. To add your address, scroll down to the bottom of .index.html, only add the key code from the api you have created after where the code says key.
3. To add your address to the map, find the comment, GOOGLE MAP. Add your address where it says, England. Done! You can find more documentation about this plugin here: http://www.birdwingfx.com/jQuery_gps/docu.html

C) JavaScript & Php - top

This theme uses javascript and php.

  1. Nivo Slider is used for the about page image slider. You don't need to do anything with the coding, but if you do you can find everything in the folder named nivo. To add your images, size them 610px x 232px and place them into your images folder. The html has been commented for you. .
    Further information can be found in the links section below.
  2. Pretty Photo is used for photo gallery. "prettyPhoto is a jQuery lightbox clone. Not only does it support images, it also support for videos, flash, YouTube, iframes and ajax." (link in credits).
  3. How to edit the photo gallery
    <a href="style/images/large.jpg" rel="prettyPhoto[pp_gal]"><img src="style/images/thumb.jpg" alt="your description" border="0" height="83" width="152" />
    1: large.jpg is your large pop-up image and small.jpg is your thumbnail. Simply add your own graphics and place them in the images > art folder.
    2: Add your description in the alt tag of the thumbnail so that it will appear when viewing the larger image. Repeat this process until you have added all ten photos.
    4: Add the class - gallast to the last thumbnail image on each row. The thumbnails are 152px x 83px in size.
    5: I have used the tag - gallery in the sortable gallery so that they don't link with your thumbnail gallery, if you decide to use both types of gallery.
  4. Contact Form
    This template uses an ajax/php form. I have explained how you can get it to forward to your email address in the html section above. All files can be found in the folder - contact.
  5. TipTip Jquery
    Used for the hover tool tip effect on text links.
    To get this to work simply add a title tag to your link and:
    a) class="link" - for a hover above tool tip
